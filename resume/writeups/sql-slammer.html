Overview

SQL Slammer (2003) is a 376-byte UDP worm that exploited a stack buffer overflow in Microsoft SQL Server 2000's resolution service (sqlservr.exe). Its payload fits entirely in a single UDP packet, making it one of the fastest-spreading worms ever observed.
